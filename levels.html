Official Levels

There are a total of 21 official levels in the game Geometry Dash. Official levels are levels that are made by RobTop himself and the only levels downloaded upon installation. This levels range from easy to demon difficulty. Only 18 are unlocked upon installation while the other 3 is not accessible yet. You would go to the main levels upon pressing the play button in the menu.



User Levels

There are more than 83 MILLION user levels that has been created in gd. User levels are levels made by players like you using the gd editor. User levels are what makes the game keep on living and not die. This is also where the hardest levels in the game reside. You can see this info in the Geometry Dash Demon List
